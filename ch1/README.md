# A Tutorial Introduction

In the first chapter of the book, it starts by introducing some of the concepts that exist in C through the use of trivial examples.

Everything from compiling a helloworld program, basic data types, standard I/O, macro definitions, etc.

Important Points:
- Text IO in C is always handled as "streams of characters", it is the responsibility of the library to make IO conform to this model - a text stream is a sequence of characters divided into lines, each containing 0 or more characters followed by a newline character.
